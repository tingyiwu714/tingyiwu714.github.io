Build personal blog with Hexo and GitHub

What is Hexo?

Hexo is a simple yet powerful static blog framework powered by Node.js.

Setup and Installation

GitHub

  1. Create a GitHub account

  2. Create a new repository named <github_username>.github.io

Git

  1. Install Git

  2. Configure your Git username/email

    • Open the Git Bash
    • Set your username:
      git config --global user.name <github_username>
    • Set your email address:
      git config --global user.email <github_email>

  3. Generate SSH Key and add to GitHub

    • From Git Bash, generate SSH key
      ssh-keygen -t rsa -C <github_email>
    • Copy the SSH key from the id_rsa.pub file, and paste to GitHub SSH key setting page
    • From Git Bash, check if the setting is successful.
      ssh git@github.com

Node.js

  1. Install Node.js

  2. From Git Bash, check if Node.js install successfully.
    node –v

  3. Check if npm install successfully.
    npm -v

Hexo

Install Hexo with npm
npm install -g hexo-cli

Start Blogging

  1. Initialize the blog in current directory
    hexo init

  2. Create a new article
    hexo new <title>

  3. Generate static file
    hexo generate

  4. Start local server
    hexo server

  5. Check the blog at http://localhost:4000

Change Theme

  1. Choose theme

  2. Clone the theme to /themes folder
    git clone <github_repo>

  3. Edit the _config.yml file in root directory to use new theme

  4. Follow the instructions on the theme’s Github read me for further setting.

  5. Start local server
    hexo server

  6. Check the blog at http://localhost:4000

Deploy GitHub Pages

  1. Edit the _config.yml file in root directory

  2. Install hexo-deployer-git
    npm install hexo-deployer-git --save

  3. Deploy the site
    hexo clean && hexo generate && hexo deploy

  4. Check the blog at http://<github_username>.github.io

Blog in Markdown

  1. Create a new article
    hexo new <title>

  2. Edit the markdown file under /source/_posts

  3. Deploy the site
    hexo clean && hexo generate && hexo deploy
